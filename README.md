
# Rainbow Delights Frontend

## Tech Stack
 
- React
- HTML
- CSS
- Javascript
- TailwindCSS




## Run Localle

Clone the project

```bash
  git clone https://github.com/Vanshika-Dargan/RainbowDelightsFrontend.git
```

Go to the project directory

```bash
  cd RainbowDelightsFrontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
